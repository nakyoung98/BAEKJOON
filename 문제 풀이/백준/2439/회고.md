# 체감 난이도

### 하

# 이유
단순한 입출력 및 구현 관련 문제  
N이 1부터 100까지기 때문에, 이중 for문을 돌려도 `100^2 = 10000`번의 계산  
1억번 계산에 1초라고 하니 큰 생각을 하지 않아도 되는 문제  

# 회고
별을 처음에 잘못 찍어서

```
*****
****
***
**
*
```

이렇게 찍었다.

```js
for (let i = 1; i <= N; i++) {
  for (let j = 1; j <= N - i; j++) {
    result += " ";
  }
  for (let k = 1; k <= i; k++) {
    result += "*";
  }
  result += "\n";
}
```
여기서 바보같이 범위 `N-i` 와 `i`를 거꾸로 썼기 때문 ㅋㅋㅋ  
잘 생각해야겠다.
